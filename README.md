# Weather App

A weather web application that fetches real-time weather data using the OpenWeather API.

## Features
- Search weather by city
- Displays temperature, humidity, wind speed
- Error handling for invalid cities

## Tech Stack
- Node.js
- Express.js
- Axios
- EJS
- OpenWeather API

## How to Run

1. Clone the repo
2. Install dependencies
3. Add `.env` file with API key
4. Run `node app.js`
